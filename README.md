## 📌 Project Overview
This project implements and compares algorithmic paradigms:
- Divide and Conquer
- Greedy Algorithms
- Dynamic Programming
- Travelling Salesman Problem (TSP)

We analyze performance using execution time and visualize results using matplotlib.

---

## ⚙️ Algorithms Implemented
1. Merge Sort (Divide & Conquer)
2. Sorting Comparison (Bubble, Merge, Quick)
3. Fractional Knapsack (Greedy)
4. 0/1 Knapsack (Dynamic Programming)
5. Travelling Salesman Problem (TSP)

---

## 📊 Performance Analysis
- Time measured using Python `time`
- Plots created using `matplotlib`
- Comparison vs input size

---

## 📈 Observations
- Merge Sort outperforms Bubble Sort as input grows
- Greedy is fast but not always optimal
- Dynamic Programming gives optimal solutions with higher memory
- TSP grows exponentially (not scalable)
